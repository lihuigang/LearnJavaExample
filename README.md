# LearnJavaExample

1.排序算法
冒泡排序:
    1.第一次排序将最大的数到队尾
    2.第二次排序将第二大的数放到倒数第二位
插入排序:
    1.分为已排序部分和未排序部分
    2.取未排序区间中的元素，在已排序区间中找到合适的插入位置将其插入
选择排序:
    1.分为已排序部分和未排序部分
    2.每次会从未排序区间中找到最小的元素，将其放到已排序区间的末尾

归并排序
    1.

    递推公式：
        merge_sort(p…r) = merge(merge_sort(p…q), merge_sort(q+1…r))
    终止条件：
        p >= r 不用再继续分解

快速排序

    递推公式：
        quick_sort(p…r) = quick_sort(p…q-1) + quick_sort(q+1… r)

    终止条件：
        p >= r
堆排序

    应用场景:
        多个有序小文件合并为一个有序大文件
